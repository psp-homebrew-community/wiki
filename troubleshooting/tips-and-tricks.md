# Tips and Tricks

Tips on how to save battery, what to do with a PSP etc

## ARK CFW Key Combos
*Some combos only work with permanent CFW*

| Key Combos | Event | Availability |
|---|---|---|
| L+R+Down+Select | Fast exit to VSH | Anytime |
| L+R+Down+Start | Fast exit to Custom Launcher | Anytime |
| L2+R2+Down+Start | Fast exit to Custom Launcher | Anytime (ARK-X) |
| L1+R1+Down+Start | Fast exit to Custom Launcher | Anytime (ARK-X VitaTV) |
| HOME | Boot OFW | Reset/Power on |
| Start | Plugins disabled | Reset/Power on/booting game |
| Select | Settings disabled | Reset/Power on/booting game |
| R | Boot Recovery | Reset/Power on |
| L | Boot DC-ARK | Reset/Power on (PSP with new cIPL) |
| Select | VSH Menu | XMB/VSH |
| L+R+Select | Force open VSH Menu | XMB/VSH |
