# Networking

- **WPA2**: Modern Wi-Fi security. The PSP only supports ancient WEP or WPA out of the box, but ARK includes a plugin so you can connect to WPA2 networks.
- **TLS**: Modern web encryption. The stock PSP browser can't load most sites today because it lacks TLS support, but developers have added it to some custom homebrew apps.
