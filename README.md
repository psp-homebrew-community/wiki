## How to add pages

Make a new markdown file and it will show up in the sidebar automatically. 

If you want the page to go into a specific category, put the file inside a folder. For example, if you make a folder called `hardware` and put `battery.md` inside it, a new "Hardware" category will appear in the sidebar with your page under it.

Make *sure* your file has a title at the top like this:

```yaml
---
layout: default
title: Your Page Title
---
```

That's about it.
